Quick help

The tool for SQL query execution in Microsoft Azure Cosmos DB

Note: Currently, support only select statement

Features:

  • double click on the collection name in the tree will create a basic SQL statement
  • databaseId separates from the collectionId by dot "."
  • databaseId not required if collectionId is unique within namespace (environment)
  • script modification is not necessary when you change environment (databaseId will be adjusted on the fly) 
  • "Fetch all" checkbox to get all documents or only first 100 (default 100)
  • last script modifications will be loaded during application start (autosave every 10 sec)

Misc:

  • Ctrl+Enter or F5 to execute script
  • Ctrl+f for search in result page
  • Ctrl+d duplicate line
  • Ctrl+/ comment and uncomment script line
  • Ctrl+z undo and Ctrl+Shift+z redo script changes
  • the script may have trailing semicolon ";"
